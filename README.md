# AI Resume Analyzer

An AI-powered Resume Analyzer built using Python, Streamlit, and Gemini API. The application helps candidates evaluate resume compatibility with job descriptions by generating ATS scores, identifying skill gaps, and providing personalized recommendations for improvement.

## Features

* Resume Upload (PDF)
* Job Description Analysis
* ATS Compatibility Scoring
* Skill Gap Detection
* Resume Improvement Suggestions
* Interview Question Generation
* AI-Powered Feedback using Gemini API

## Tech Stack

* Python
* Streamlit
* Gemini API
* Prompt Engineering
* PDF Processing
* Pandas

## Project Overview

The AI Resume Analyzer compares a candidate's resume with a target job description and provides detailed feedback. It uses Large Language Models (LLMs) through Gemini API to analyze resume content, evaluate skill alignment, and generate actionable suggestions.

## Workflow

1. Upload Resume (PDF)
2. Paste Job Description
3. Extract Resume Content
4. Analyze Skills and Experience
5. Calculate ATS Match Score
6. Generate Improvement Suggestions
7. Create Interview Questions

## Installation

Clone the repository:

```bash
git clone https://github.com/ranjanmohit521/ai-resume-analyzer-gemini.git
cd ai-resume-analyzer-gemini
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
