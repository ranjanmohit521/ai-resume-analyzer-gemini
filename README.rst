# AI Resume Analyzer

::

```
An AI-powered Resume Analyzer built using Python, Streamlit, and Gemini API.
```

The application helps candidates evaluate resume compatibility with job descriptions by generating ATS scores, identifying skill gaps, and providing personalized recommendations.

# Features

* Resume Upload (PDF)
* ATS Score Analysis
* Resume vs Job Description Matching
* Skill Gap Detection
* Resume Improvement Suggestions
* Interview Question Generation
* AI-Powered Feedback using Gemini API

# Tech Stack

* Python
* Streamlit
* Gemini API
* Prompt Engineering
* Pandas
* PDF Processing

# Project Overview

The AI Resume Analyzer compares a candidate's resume against a target job description and provides detailed AI-generated feedback.

The system can:

* Analyze resume content
* Identify matching and missing skills
* Generate ATS compatibility scores
* Suggest resume improvements
* Create personalized interview questions

# Workflow

1. Upload Resume (PDF)
2. Paste Job Description
3. Extract Resume Content
4. Analyze Skills and Experience
5. Calculate ATS Match Score
6. Generate Recommendations
7. Create Interview Questions

# Installation

::

```

```

::

```

```

::

```
pip install -r requirements.txt
```

::

```

```
